# 習慣トラッカーアプリ

Next.js App Router、Tailwind CSS、TypeScriptを使用したモダンで清潔感のある習慣化アプリです。

## 特徴

- 📊 ストーリーズ映えする進捗グラフ
- 🎯 習慣の追跡と連続記録
- 📱 レスポンシブデザイン
- 🎨 モダンで清潔感のあるUI

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いて確認してください。

## 技術スタック

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **React 18**
